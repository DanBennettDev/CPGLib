Detailed Description

Engine: Main interface for controlling and calculating the CPG network.

Threadsafe - step(), getEvents() etc. can be run in one thread, while controlling the system using the other functions from another thread

Some functions are immediately actioned, some are queued and actioned on calling doQueuedActions(). If queued this is commented
